3. 支持统计类数据输出：sum, count, avg……

7. 接口状态管理（研发流程的管控）

    - 对接：经过mock过的接口
    - 调试：开发环境跑过的接口
    - 稳定：开发环境N个测试数据测试通过的接口
    - 测试：测试环境开始跑的接口
    - 发布：测试环境N个测试数据测试通过的接口
8. 数据统计
  - 增加接口过程管理的数据统计

  - 项目接口测试统计
  - 并发测试环境数据采集和展示
  - 并发测试性能数据采集和展示
